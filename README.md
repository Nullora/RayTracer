# Raytracer
Made to practice C and SDL2. Heavily inspired from [Daniel Hirsch](https://www.youtube.com/watch?v=2BLRLuczykM) and google documentation!
# Why I made it
I was always facinated by 3d graphics and really wanted to get into it. So i started with simple 2d raytracing as an introduction! I tried to start directly with
Vulkan and it went south so i started with 2d CPU based SDL2. I chose CPU only so im forced to optimize my code as much as possible.
You can read the comments on my code to know why I made each decision. Again thanks to Daniel Hirsch for the detailed explanation.
# Note
I only posted this on GitHub to document it for others who might want to get into 3d graphics and don't wanna jump in the deep water yet.
This is not meant to be polished and not meant for any actual use other than messing around for fun!
